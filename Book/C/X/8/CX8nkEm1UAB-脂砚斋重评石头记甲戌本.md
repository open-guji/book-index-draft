# 脂砚斋重评石头记甲戌本
ID：CX8nkEm1UAB

## 基本信息
- **作品名**：脂砚斋重评石头记（甲戌本）
- **收录于**：脂评系列丛书 / 早期手抄本
- **作者**：曹雪芹（著）；脂砚斋（评）
- **出版年份**：清乾隆十九年（1754年）定稿，现存约为乾隆晚期抄本
- **现藏于**：上海博物馆（2005年自美国购回）
- **页数**：现存16回，共四册（第1-8回，13-16回，25-28回）

## 介绍
乾隆十九年（1754年）甲戌抄本，简称“**甲戌本**”。它是《红楼梦》现存传世版本中**存世最早**、**保留脂批最多**、**最接近原著面貌**的珍稀抄本。虽然该本仅存16回，但其正文前的《凡例》以及密集的朱红色批注（脂批），为研究曹雪芹的创作意图、家族背景及小说佚失结尾提供了极其关键的线索。

## 收藏历史
1. **清末**：由大兴**刘铨福**收藏，册内有其大量题跋。
2. **民国**：1927年，**胡适**以重金（三十大洋）从上海地摊购得。胡适凭借此本撰写了《考证红楼梦的新材料》，确立了新红学范式。
3. **流转**：1948年胡适将其带往美国，后长期存放于**康奈尔大学图书馆**。
4. **回归**：2005年，在国家文物局的支持下，由**上海博物馆**斥资400万美元购回。

- [舒元炜序本（舒序本）](bid:\\CX9X9nAnzrj): 乾隆五十四年（1789年）舒元炜作序并整理，现存前四十回，带有大量具有时代特色的舒评。
- [郑振铎藏残本（郑藏本）](?): 仅存第二十三、二十四回，文字极古，与甲戌本互有印证，极具版本学价值。
