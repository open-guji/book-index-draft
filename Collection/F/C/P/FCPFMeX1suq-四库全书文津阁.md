# 四库全书文津阁
ID：FCPFMeX1suq

## 基本信息
- 作品名：钦定四库全书（文津阁本）
- 收录于：四库全书
- 作者：纪昀等奉敕编纂
- 出版年份：清乾隆四十九年（1784年）抄写完成
- 现藏于：中国国家图书馆
- 页数/册数：36,304册，6,144函，分装于128个书架

## 介绍
文津阁《四库全书》是清代乾隆年间编纂的《四库全书》七部之一。它依据文渊阁本校对并抄写，包含了经、史、子、集四部，收书3,503种。文津阁本是“北四阁”（文渊、文源、文溯、文津）中较晚抄写完成的一部，因此在编纂过程中对前三部中出现的错漏进行了补正。该版本最大的特点是“馆、阁、匣、书”保存最为完整，不仅书籍本身保存完好，连原本的存放书架和木匣也基本保存原貌，是目前唯一一套原架、原函、原书一体保存的《四库全书》。

## 收藏历史
1. **乾隆时期**：1784年（乾隆四十九年）抄写完毕，入藏热河行宫（今承德避暑山庄）文津阁。
2. **民国初期**：1913年，当时的民国政府教育部将文津阁本自承德运至北京。
3. **入藏京师图书馆**：1915年，正式移交至京师图书馆（中国国家图书馆前身）收藏。
4. **战时流转**：1930年代，为避战火，曾与大量古籍一同南迁。
5. **现状**：现完整收藏于中国国家图书馆（国家典籍博物馆），作为该馆的“四大镇馆之宝”之一。
