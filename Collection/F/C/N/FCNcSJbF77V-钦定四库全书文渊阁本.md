# 钦定四库全书（文渊阁本）
ID：FCNcSJbF77V

## 基本信息
- 收录于： 
- 作者：纪昀（总纂官）等奉敕编纂
- 出版年份：清乾隆四十七年（1782年）完成
- 现藏于：台北故宫博物院
- 页数/册数：3,461种，36,304册，约7.9亿字

## 介绍
《四库全书》是在乾隆皇帝的主持下，由纪昀等360多位高官、学者编撰，近4000人抄写，耗时十年完成的丛书。它是中国历史上规模最大的丛书，分经、史、子、集四部，故名“四库”。

文渊阁本是《四库全书》的第一部正本，也是“北四阁”（文渊阁、文源阁、文津阁、文溯阁）之首。其装帧精美，采用“经青、史赤、子子、集灰”的绢面颜色来区分四部。

## 包含书籍
- [经部](ID-经部.md)：收录儒家经典及相关研究著作（如《周易》、《诗经》等）。
- [史部](ID-史部.md)：收录正史、编年史、地理、职官等史料。
- [子部](ID-子部.md)：收录诸子百家、天文算法、艺术、谱录、农医等。
- [集部](ID-集部.md)：收录历代文学作品、诗文集等。

## 收藏历史
1. **乾隆时期**：书成后入藏紫禁城内的文渊阁。
2. **抗日战争时期**：为躲避战火，文渊阁本随故宫文物南迁。
3. **1940年代末**：随国民政府迁往台湾。
4. **现状**：现由台北故宫博物院妥善保管，并已进行数字化和影印出版。
