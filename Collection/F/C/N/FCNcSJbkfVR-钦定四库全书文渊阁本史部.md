# 钦定四库全书（文渊阁本）史部
ID：FCNcSJbkfVR

## 基本信息
- 收录于：钦定四库全书（文渊阁本）
- 作者：纪昀（总纂官）等奉敕编纂
- 出版年份：清乾隆四十七年（1782年）
- 现藏于：台北故宫博物院
- 页数/册数：约 560 余种

## 介绍
史部为《四库全书》“经、史、子、集”四部之第二大部，主要收录中国历代史学著作。文渊阁本史部书籍的封面装帧统一使用**红色（赤色）绢面**，寓意“史鉴如火”或对应南方属火之色。其收录范围广泛，不仅包含正史，还涵盖了地理、职官、政书、目录及金石等各类史料，是研究中国古代社会、政治、经济、地理的重要文献库。

## 包含书籍

- [编年类](ID-编年类.md)：以时间为序的史书，如《资治通鉴》。
- [纪事本末类](ID-纪事本末类.md)：以事件为中心的史书。
- [别史类](ID-别史类.md)：不属于正史但具有重要史料价值的史书。
- [载记类](ID-载记类.md)：记录割据政权或边疆地区的史学著作。
- [诏令奏议类](ID-诏令奏议类.md)：历代皇帝诏令及臣下奏疏。
- [传记类](ID-传记类.md)：各类型的人物传记、家谱等。
- [史钞类](ID-史钞类.md)：史书的摘抄与精简本。
- [史评类](ID-史评类.md)：对历史事件或人物的评论与分析。
- [政书类](ID-政书类.md)：记录典章制度的著作，如《通典》。
- [目录类](ID-目录类.md)：收录历代官修、私修书目。
- [地理类](ID-地理类.md)：涵盖总志、都会郡县、名胜、方志等。
- [职官类](ID-职官类.md)：记录官制、秩禄等。
- [金石类](ID-金石类.md)：收录考证青铜器、碑刻的著作。
